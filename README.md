
# Chatter-A social media Website

A social-media platform where anyone can chat with anyone and also anyone can see any person post


## Requirements

 - NodeJS 
 - MongoDB
 - Redis Server
## Installation

Install all packages and dependencies

```bash
  npm install

```
Run the server in development mode

```bash
  npm run dev

```
